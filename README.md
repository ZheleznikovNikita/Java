# Java Labs Collection

This repository contains a collection of Java laboratory assignments covering fundamental programming concepts.

## Repository Structure

The project is organized into multiple lab directories:

```
/workspace
├── Java_Lab1/     # Basic I/O, arithmetic operations, and simple tasks
├── java_lab2/     # (Lab 2 contents)
├── Java_Lab3/     # (Lab 3 contents)
├── Java_Lab4/     # (Lab 4 contents)
├── Java_Lab5/     # (Lab 5 contents)
└── Java_Lab6/     # (Lab 6 contents)
```

Each lab directory contains:
- `src/` - Source code files (.java)
- `.idea/` - IntelliJ IDEA project configuration
- `.gitignore` - Git ignore rules

## Lab Topics

### Java Lab 1
Basic programming tasks including:
- Greeting and user input
- Temperature conversion
- Arithmetic operations
- Sum of digits calculation
- Circle circumference and area
- Distance between points
- Time conversion (minutes to hours)
- Number reversal

### Java Labs 2-6
Additional laboratory assignments building on fundamental concepts.

## Requirements

- **Java Development Kit (JDK)**: Version 8 or higher
- **IDE**: IntelliJ IDEA (recommended) or any text editor with Java support

## How to Compile and Run

### Using Command Line

Navigate to the desired lab directory:

```bash
cd Java_Lab1/src
```

Compile all Java files:

```bash
javac *.java
```

Run the main program:

```bash
java Main
```

### Using IntelliJ IDEA

1. Open IntelliJ IDEA
2. Select `File > Open` and choose the lab directory (e.g., `Java_Lab1`)
3. Wait for the project to index
4. Right-click on `Main.java` and select `Run 'Main.main()'`
